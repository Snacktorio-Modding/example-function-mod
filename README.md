# Example Function Mod
An example mod for [Snacktorio](https://store.steampowered.com/app/1902940/Snacktorio/), showing how you can add new functionality to the game.  
https://steamcommunity.com/sharedfiles/filedetails/?id=3726345843

In this example, we add an big 'spring' jump when pressing spacebar, emulating the jump you get from springboards in the game. Look out below!

Part of the "Creating Custom Content!" guide for Snacktorio:
https://steamcommunity.com/sharedfiles/filedetails/?id=3726274367
